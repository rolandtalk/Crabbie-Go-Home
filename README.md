# Crabbie Go Home

A small 2D browser game where Crabbie the horseshoe crab dodges moving trap nets
and tries to reach Sweet Home.

## Run it

Open `/Users/rolandtalkonmini/Documents/Crabie Go Home/index.html` in a browser.

## Controls

- Arrow keys: move Crabbie
- Space or Enter: start / continue after a win
- R: restart the game
